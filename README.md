# AI Revolution - Beautiful AI Website

A stunning, fully animated, and responsive website showcasing AI capabilities built with React, TypeScript, Tailwind CSS, and Framer Motion.

## 🚀 Features

- **Beautiful Design**: Modern, glass-morphism effects with gradient animations
- **Fully Responsive**: Works perfectly on all device sizes
- **Smooth Animations**: Advanced micro-interactions and page transitions
- **AI Theme**: Futuristic AI-inspired design with animated elements
- **Interactive Components**: Live chat demo, feature showcases, and more
- **TypeScript**: Full type safety for better development experience

## 🛠️ Technologies Used

- **React 18** - Modern React with hooks
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Advanced animations library
- **Lucide React** - Beautiful icon library
- **React Router** - Client-side routing

## 📦 Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd ai-website
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 🎨 Customization

### Colors
The theme uses custom AI-inspired colors defined in `tailwind.config.js`:
- `ai-blue`: Primary blue color
- `ai-purple`: Secondary purple color
- `ai-pink`: Accent pink color
- `ai-cyan`: Cyan highlight color

### Animations
Custom animations are defined in the Tailwind config:
- `float`: Gentle floating animation
- `gradient`: Animated gradient background
- `pulse-slow`: Slow pulsing effect

## 🚀 Deployment

### Build for Production
```bash
npm run build
```

The build files will be in the `build` folder, ready for deployment.

### Deploy to Vercel
1. Push your code to GitHub
2. Connect your repository to Vercel
3. Deploy automatically

### Deploy to Netlify
1. Run `npm run build`
2. Upload the `build` folder to Netlify
3. Configure your custom domain
